# 魔兽世界插件管理工具

一个基于 `WPF` 和 `.NET 8` 的本地桌面工具，用于管理魔兽世界插件目录、备份插件配置，以及在清理插件时保留指定插件目录。

## 功能简介

- 自动检测或手动选择魔兽世界安装目录
- 一键清理 `Cache`、`Fonts`、`Interface`、`WTF`
- 清理前可自动创建 ZIP 备份
- 配置备份：按账号读取插件配置并选择性备份
- 备份记录管理：支持恢复、删除，双击记录可打开备份文件位置
- 设置页：可配置一键清理时需要保留的插件目录

## 当前清理规则

- “保留插件”只作用于 `Interface/AddOns`
- 不会自动保留 `WTF` 配置
- 如果需要保留插件配置，请使用“配置备份”选项卡先进行备份

## 使用方式

1. 启动程序后，先确认魔兽世界安装目录是否识别正确。
2. 在“清理插件”页选择要清理的文件夹，并决定是否创建备份。
3. 如果希望清理时保留某些插件，先到“设置”页搜索并勾选对应插件。
4. 如果需要保留插件配置，请到“配置备份”页按账号选择并备份。
5. 如需恢复备份，可在“配置备份”页选择备份记录进行恢复。

## 构建环境

- Windows
- .NET SDK 8.0

## 本地构建

```powershell
dotnet build
```

## 项目结构

- `MainWindow.xaml`：主界面
- `MainWindow.xaml.cs`：主要业务逻辑
- `App.xaml` / `App.xaml.cs`：应用入口
- `WowUtils.csproj`：项目配置

## 说明

该工具会对本地魔兽世界目录执行删除、压缩和恢复操作。执行清理或恢复前，建议先确认路径和账号信息，并保留必要备份。
